## System Architecture Summary

The project is a full-stack Todoist Agent application with the following architecture:

**Tech Stack:**
- **Backend**: Convex (database + serverless functions)
- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: TailwindCSS with ChatGPT grey theme
- **Authentication**: Convex Auth with Firebase integration
- **AI Integration**: Claude AI (Anthropic SDK) for task management

**Architecture Flow:**
```
React Frontend → Convex Backend → Claude AI (Anthropic SDK) → Convex Database
                      ↓
            Real-time Updates & Authentication
```

**Key Components:**
- **AI Agent System**: Located in `convex/ai/` with tool registry and subagents
- **Todoist Integration**: OAuth flow and API integration in `convex/todoist/`
- **User Management**: Firebase-based authentication with user profiles
- **Chat Interface**: Real-time conversation management with AI agent
- **Task Management**: Full CRUD operations for Todoist tasks

**Project Structure:**
- Main active project: `ea-ai-main2/ea-ai-main2/`
- Backend logic: `convex/` folder
- Frontend components: `src/` folder
- Development logs: `updates/` folder
